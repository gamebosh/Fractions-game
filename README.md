# GameBosh Fraction Quest

A colourful educational fractions game for children.

- 240 progressive levels
- More than 1,200 challenges
- Visual explanations, hints, animations, sounds, and saved progress
- Proper, improper, mixed, and equivalent fractions
- Comparing and all four fraction operations

## Run locally

```bash
npm install
npm run dev
```
